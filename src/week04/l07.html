A pure function is a function that adheres to the following rules:

  • The return value of a pure function should only depend on the values provided as arguments. it does not rely on values from somewhere else in the program (DONT USE GLOBALS)
  • There are no side-effects. A pure functoin does not change any values or data elsewhere in the program. It only makes non-destructive data transformations and returns new values, rather than altering any of the underlying data.
  • Referential transparency. Given the same arguments, a pure function will always return the same result.
