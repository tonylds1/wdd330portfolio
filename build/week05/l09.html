Web Storage Vs Cookies

  • Information stored is not shared with the server on every request
  • Information is available in multiple windows of the browser (but only if the domain is the same)
  • Storage capacity limit is much larger than the 4KB limit for cookies (there is no actual limit in the specification, but most browser have a limit set at %GB per domain.)
  • Any data stored does not autimatically expire as it does with cookies. This potentially makes cookies a better choice for something like showing a popup once a day

There is also a sessionStorage object that works in the same way, although the data is only saved for the current session
